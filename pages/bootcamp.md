This bootcamp is based on the Python course I teach every semester at the Weizmann Institute of Science
and the various Python courses I taught at corporations such as Intel, Qualcomm, Bezeq International and man more.

## Who is this course for?

* A student who never took a programming course but would like to be able to start programming.
* A student who took a programming course, but back then only wanted to get a grade and now wants to actually do some programming.
* Someone who have some experience in programming, but feels that the "programming with AI" is overwhelming.
* Someone who want to increase his/her chances of finding a job now that "everyone uses AI".

## Major topics in the bootcamp

* Version control using git and GitHub
* Python programming using AI tools such as Claude Code, Google Gemini (Anti-gravity), Open-AI Codex (ChatGPT), GitHub (Microsoft) Copilot.
* Python syntax.
* Testing.
* CI/CD (Continuous Integration / Continuous Delivery)
* Front-end programming.
* Linux command line.

## Content

* Git
* GitHub

    * Pages
    * Issues
    * Pull-Request
    * Actions
    * Co-pilot
* GitLab
    * Merge-Request
    * Pipeline

* Markdown

* Front-end creation
* Python programming
* APIs, JSON
* File formats: JSON, YAML, TOML, XML, HTML, INI, etc.
* Linux command line and basic level of system administration.
* Machine learning. Analyzing various data-sets.

## Expenses

Besides the fee of the course itself you'll have to pay for some external services.
Most of these services have a limited availability free of charge.
For the duration of the course you might want to upgrade one or more of these services to a paid account.

Some of the prices as I saw in July 2026.

* One or more AI systems
    * [GitHub co-pilot](https://github.com/features/copilot/plans): $10-40/month.
    * [ChatGPT](https://chatgpt.com/pricing) and [Codex](https://chatgpt.com/codex/pricing/): $25/month.
    * [Google Gemini + Antigravity](https://gemini.google/subscriptions/): $25/month.
    * [Anthropic Claude Code](https://claude.com/pricing): $17/month.

* Renting a Virtual Private Server (VPS) $5/month or more.
    * [Linode](https://www.linode.com/pricing/)
    * [Digital Ocean](https://www.digitalocean.com/pricing)
