We would like to increase the number of people who enjoy the values of the events we organize in English.

## What to promote?

The events in the [Code Mavens](https://www.meetup.com/code-mavens/) Meetup group that are also listed on the [live events page](https://live.code-maven.com/).

The video recordings of these events in the [YouTube channel](https://www.youtube.com/@CodeMaven).


## How and Where to promote?

* Create better banners for the Meetup groups
* Improve the descriptions.

* Create better thumbnails for the YouTube videos
* Add better description to the videos.


* [Online events](https://events.code-maven.com/) page.
